Write down styling attributes you can use while creating table and explain what they do. There are several examples of table styling attributes. The border attribute allows you to detail the border width, type (solid, dotted) and color of the table, rows and cells. You can also use width and height to set the size of the table. You can also align the text using text-align, center left or right. Padding is used to set space between text or images in a table and table borders. You can also set the background color of the table. There are many other options, but these are a few of them.
